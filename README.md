# SQL Server Simulation 6 - AdventureWorks Trigger and Auditing Lab

This repository contains the SQL Server simulation project for Enterprise Trigger Development and Database Auditing using the AdventureWorks2022 database.

## Project focus
The project includes:
- audit environment setup
- trigger-based auditing logic
- recursive trigger demonstration for Task 5
- deployment and validation scripts
- reporting queries
